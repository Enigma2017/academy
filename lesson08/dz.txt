Домашнее задание:
1.
Есть сайт, на котором человек регистрируется и может указать, что у него есть собака.
В зависимости от этого получим такие варианты:
var user1 = { name: "Harry", dog: null };
var user2 = { name: "David", dog: { name: "Lucky" } };
После регистрации владельцам собаки предлагается скидка,
за которую отвечает вот такой код:
function notify() {
    console.log("You have a dog! Use this code for 50% off: UG4H6QE");
}
function checkDogOwner(user, callback) {
    if (user && user.dog) {
        callback();
    }
}
// on registration success
checkDogOwner(user, notify);

Вам нужно покрыть тестами работу функции checkDogOwner.


2.
Написать тесты для проверки работы метода splice, который есть у массивов.
То есть, в данном случае мы пишем только тесты, которые будут работать с массивами и активно использовать метод splice.
Проверить нужно каждую мелочь работы этого метода:
есть ли метод вообще, правильные ли переданы параметры
(напомню, что некоторые параметры не обязательные),
какой результат возвращает метод, правильно ли он отрабатывает в принципе.


PS: в codepen можно нажать на settings --> javascript и подключить все необходимые файлы jasmine.
ссылки можно найти тут: https://cdnjs.com/libraries/jasmine
